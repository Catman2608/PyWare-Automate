## How to compile
For windows: Use Auto Py To Exe
For macOS: Use Pyinstaller
For Linux: I don't have a Linux machine so if you want to use it on Linux compile the file yourself
## What is this?
PyWare Automate is a cross-compatible recorder and playback tool that exports files as AutoHotKey syntax.
Basically this allows macOS and possibly Linux users to run AHK scripts for the first time.
